Guitar Tone Dataset

An open dataset of 748 researched guitar tones — the original gear and amp settings behind famous riffs, with source citations. CC BY 4.0.

🎸 Adapt any tone to YOUR rig — GuitarToneAdapt ★ View on GitHub
What's in it
Each of the 748 entries has: song, artist, genre, era, tone_type, amp_settings (gain/bass/mid/treble/presence/reverb), original_gear (guitar, amp, pedals), effects, pickup, confidence, sources, and a url. Download: guitar-tones.json.

Explore the tones

This dataset powers GuitarToneAdapt — a free web app that re-dials any famous song's guitar tone for the specific guitar, amp and pedals you own. Browse the tones:

  • All collections — by scene, genre & language
  • Anime, J-rock & game guitar tones
  • Worship guitar tones

  • Amp settings cheat sheets · Tone data report

How to use / cite

Useful for tone-matching tools, music ML, and "what amp settings for <song>" lookups. Attribution: GuitarToneAdapt (guitartoneadapt.com).
